# Current Runtime Contracts

These schemas define the current executable toolkit boundary that automation is
expected to rely on today.

Covered contracts:

- `auth-check-result`
- `fetch-templates-result`
- `view-selected-template-result`
- `view-preview-dump-result`
- `render-newsletter-input`
- `render-newsletter-result`
- `clone-template-result`
- `publish-rendered-draft-result`

The schemas are intentionally focused on the current runtime path, not the full
planned next-state architecture.
